# Vuforia engine
Download de Unity versie van deze site:
https://developer.vuforia.com/downloads/sdk

Dan zou je nu de package moeten hebben staan.
Ga naar Assets -> Import package -> Custom Package
ALs je drukt op Import krijg je een promt:
- Druk hier op update.

Nu zou Vuforia moeten toegevoegd zijn in je Unity project.


# Image target aanmaken
In je scene, verwijder je als eerste de "Main camera", en voeg je de "AR camera" toe.
Deze kan je vinden door rechtermuisknop te doen in je scene, naar "Vuforia Engine" te gaan, en voor AR camera te kiezen.

Daarna kan je via dezelfde methode een Image target toevoegen
